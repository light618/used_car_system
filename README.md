# 二手车管理系统

基于PHP开发的二手车交易管理系统，支持微信小程序、H5和PC端三合一。

## 技术栈

- **后端**: PHP 7.4+ / PHP 8.0+
- **数据库**: MySQL 5.7+ / MySQL 8.0+
- **前端框架**: ThinkPHP 6.0 / Laravel (可选)
- **小程序**: 微信小程序原生开发

## 功能模块

### 1. 车辆管理
- 车辆信息发布
- 车辆分类管理
- 车辆搜索筛选
- 车辆详情展示
- 车辆图片管理

### 2. 用户管理
- 用户注册/登录
- 会员管理
- 用户认证
- 用户收藏/浏览记录

### 3. 订单管理
- 预约看车
- 订单创建
- 订单支付
- 订单状态跟踪

### 4. 后台管理
- 车辆审核
- 用户管理
- 订单管理
- 数据统计
- 系统配置

### 5. 其他功能
- 消息通知
- 评价系统
- 优惠券管理
- 数据报表

## 项目结构

```
used-car-system/
├── api/              # API接口
├── admin/            # 后台管理
├── miniprogram/      # 小程序代码
├── config/           # 配置文件
├── database/         # 数据库文件
├── public/           # 公共资源
├── vendor/           # 第三方库
└── README.md
```

## 安装说明

1. 克隆项目
2. 配置数据库连接
3. 导入数据库文件
4. 配置微信小程序参数
5. 部署到服务器
