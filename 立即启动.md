## 📦 安装PHP（只需一次）

### macOS安装命令：
```bash
brew install php
```

安装完成后，PHP会自动添加到系统PATH中。

## 🚀 启动服务器

安装PHP后，执行以下命令启动：

```bash
cd /Users/yiche/used-car-system
bash 启动服务器-手动.sh
```

或者直接运行：
```bash
php -S localhost:8080 -t public router.php
```
