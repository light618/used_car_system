## 三、完整流程测试

### 第一步：创建门店
1. 登录总部管理员账号
2. 进入"门店管理"
3. 点击"新增门店"
4. 填写门店信息（编码、名称、位置、电话）
5. 创建成功后会提示默认账号信息

### 第二步：录入车源
1. 使用门店录入员账号登录（门店编码_luru / 123456）
2. 进入"新增车源"
3. 填写完整的车源信息
4. 上传车辆照片（至少3张）
5. 上传绿本照片（至少1张）
6. 提交审核

### 第三步：审核车源
1. 使用门店管理员账号登录（门店编码_admin / 123456）
2. 进入"待审核车源"
3. 查看车源详情
4. 点击"通过"或"驳回"
5. 如果驳回，需要填写驳回原因

### 第四步：授权车源
1. 使用总部管理员账号登录
2. 进入"车源管理"
3. 查看已审核通过的车源
4. 点击"授权"按钮
5. 选择目标门店（可多选）
6. 确认授权

### 第五步：查看授权车源
1. 使用被授权门店的管理员账号登录
2. 进入"车源管理"
3. 筛选选择"全部"或"他店"
4. 可以看到授权过来的车源，显示"他店"标签和原门店联系电话
